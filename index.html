Claude(클로드)가 뭐예요?

한 줄 요약: ChatGPT 같은 AI 챗봇인데, 긴 글·문서 정리·정중한 글쓰기에 특히 강한 친구예요.

Claude는 미국 회사 Anthropic(앤트로픽)이 만든 대화형 AI예요. 주소는 claude.ai 하나만 기억하면 됩니다. 웹브라우저로도 쓰고, 휴대폰 앱(아이폰·안드로이드)으로도 써요.

ChatGPT랑 뭐가 달라요?

둘 다 "질문하면 답해주는 AI"라는 점은 같아요. 큰 그림만 보면 됩니다.

항목ChatGPTClaude
잘하는 것이미지 생성, 폭넓은 만능형긴 문서 읽기·요약, 다듬어진 글쓰기, 표·문서 만들기
말투 느낌활발하고 자유로움차분하고 정중함 — 공문·가정통신문에 잘 맞음
특징 기능GPTs, 이미지(DALL·E)아티팩트(옆 화면에 결과물), 프로젝트, 스킬
파일 첨부가능가능 — PDF·한글캡처·엑셀 등 잘 읽음
🔄 ChatGPT 써봤다면

사용법은 거의 똑같아요. 입력창에 질문 → 엔터. 새로 배울 건 "모델 이름"과 "아티팩트·프로젝트" 같은 Claude만의 단어 몇 개뿐이에요. 이 가이드가 그것만 콕 집어줄게요.

요금제 — 무료로 시작해도 돼요

요금제가격(대략)이런 분께
Free(무료)0원처음 맛보기. 하루 사용량 제한 있음, 똑똑한 모델은 조금만 쓸 수 있음
Pro월 2만 원대업무에 본격 활용하는 선생님. 사용량 넉넉, 프로젝트·아티팩트 풍부
Max월 10만 원대~하루 종일 헤비하게 쓰는 경우
💡 꿀팁

우선 무료로 가입해서 며칠 써보고, "이거 매일 쓰겠다" 싶을 때 Pro로 올리세요. 무료로도 이 가이드 내용 대부분을 체험할 수 있어요.

시작하기 & 화면 둘러보기

가입은 3분. 화면은 딱 세 군데만 기억하면 돼요.

가입·로그인

  1. claude.ai 접속 브라우저 주소창에 claude.ai 입력. 휴대폰은 앱스토어/플레이스토어에서 "Claude" 검색해 설치.
  2. 구글 계정으로 로그인 "Continue with Google"을 누르면 가장 빠르고 안전해요. 이메일로 가입도 가능.
  3. 이름·간단 정보 입력 화면 안내대로 따라가면 끝. 첫 대화창이 바로 열려요.
✅ 학교 계정 주의

교육청·학교 구글 계정(@xxx.go.kr 등)은 보안 정책으로 외부 로그인이 막혀 있을 수 있어요. 그럴 땐 개인 지메일로 가입하세요.

화면 구성 한눈에

처음 보면 복잡해 보이지만, 실제로 자주 쓰는 곳은 세 군데예요.

📝

① 가운데 입력창

여기에 질문을 적고 엔터. 모든 게 여기서 시작돼요. 파일·이미지 첨부 버튼(📎)도 여기 있어요.

📚

② 왼쪽 사이드바

지난 대화 목록, 새 대화(＋) 버튼, 프로젝트, 설정 메뉴가 모여 있어요.

🔽

③ 모델 선택 버튼

입력창 근처에 모델 이름(예: Sonnet)이 보여요. 눌러서 똑똑한 모델로 바꿀 수 있어요.

💡 꿀팁

왼쪽 사이드바가 안 보이면 화면 왼쪽 위 ☰ 아이콘을 누르세요. 휴대폰에서는 기본적으로 숨겨져 있어요.

3

첫 대화 — 기본 사용법

"질문 잘하는 법"이 실력의 90%. 어렵지 않아요.

그냥 말 걸듯이 쓰면 돼요

입력창에 적고 Enter. 줄바꿈은 Shift+Enter예요. 답이 마음에 안 들면 그냥 이어서 "더 짧게", "더 정중하게", "표로 정리해줘"라고 말하면 고쳐줘요.

좋은 질문의 4가지 재료

이 네 가지를 넣으면 답이 확 좋아져요. 다 넣을 필요는 없고, 생각나는 대로요.

👤 누구로서

"초등 5학년 담임 교사인데"

🎯 무엇을

"체험학습 가정통신문을 써줘"

📋 조건

"5월 20일, 경복궁, 회비 1만원 포함"

✨ 형식

"정중한 안내문 톤으로, A4 반 장"

예시 프롬프트 초등학교 5학년 담임 교사예요. 체험학습 가정통신문을 써주세요. 날짜는 5월 20일, 장소는 경복궁, 준비물은 도시락과 물, 회비는 1만 원이에요. 학부모님께 보내는 정중한 안내문 톤으로, A4 반 장 분량으로 부탁해요.

꼭 알아둘 기본 동작

  • 파일·이미지 첨부 📎 버튼으로 PDF·엑셀·한글 캡처·사진을 올리고 "이거 요약해줘 / 표로 만들어줘" 가능.
  • 새 대화 시작 주제가 바뀌면 ＋ 새 대화를 누르세요. 한 대화에 여러 주제를 섞으면 AI가 헷갈려요.
  • 답변 다시 받기 답 아래 ↻(다시 생성) 버튼으로 다른 버전을 받을 수 있어요.
  • 복사하기 답변 아래 복사 아이콘으로 한글·메일에 바로 붙여넣기.
⚠️ AI는 가끔 틀려요

날짜·숫자·규정·인용은 그럴듯하게 지어낼 수 있어요. 공문·평가에 쓰기 전엔 사람이 꼭 확인하세요. "확실하지 않으면 모른다고 해줘"라고 덧붙이면 도움이 돼요.

다음: 모델 고르기 ↓

모델 고르기 — Opus · Sonnet · Haiku

"모델"은 AI의 두뇌 종류예요. 똑똑함 ↔ 속도를 고르는 일이라고 생각하세요.

입력창 근처 모델 이름 버튼을 누르면 목록이 나와요. 이름이 어렵게 느껴지면 아래 비유 한 줄만 기억하세요.

모델비유언제 써요?
Opus🧠 깊이 고민하는 박사님어려운 글쓰기, 기획안, 복잡한 자료 분석. 가장 똑똑하지만 조금 느리고 사용량 소모 큼
Sonnet⚖️ 일 잘하는 만능 직원대부분의 평소 업무에 추천. 똑똑함과 속도 균형이 좋아요
Haiku⚡ 빠릿한 신입간단한 요약, 맞춤법, 짧은 답. 가볍고 빠름
💡 그냥 이렇게 하세요

고민되면 기본값(보통 Sonnet) 그대로 쓰면 돼요. 결과가 아쉬우면 그때 Opus로 한 단계 올려서 다시 시켜보세요. 모델은 대화 도중에도 언제든 바꿀 수 있어요.

⚠️ 무료 사용자는

가장 똑똑한 Opus는 무료에서 횟수가 적거나 안 보일 수 있어요. 평소엔 Sonnet, 중요한 한 방엔 Opus — 이 정도면 충분해요.

※ 모델 이름 뒤 숫자(예: Opus 4.8)는 버전이에요. 숫자가 클수록 최신·향상판이라고 보면 됩니다.

응답 스타일(Styles) — 말투 바꾸기

매번 "정중하게 써줘"라고 안 적어도, 기본 말투를 정해둘 수 있어요.

스타일은 Claude의 답변 톤·길이를 미리 정해두는 기능이에요. 입력창 근처(또는 모델 버튼 옆)의 스타일 버튼에서 고릅니다.

스타일느낌이럴 때
표준(Normal)기본평소 그대로
간결(Concise)짧고 핵심만빠른 답, 요점만 보고 싶을 때
설명형(Explanatory)자세히 풀어서개념을 배우거나 학생 설명 자료 만들 때
격식(Formal)정중·공식적공문, 학부모 안내, 보고서

나만의 스타일도 만들 수 있어요

  1. 스타일 버튼 → "스타일 만들기/생성" 클릭
  2. 원하는 말투 설명 예: "항상 존댓말, 학부모 안내문 톤, 어려운 용어는 풀어서, 끝에 한 줄 요약"
  3. 저장 후 선택 이제 그 대화는 늘 그 톤으로 답해요
💡 차이 정리

스타일 = 말투/형식, 지침(7번) = 나에 대한 정보. 둘 다 "매번 안 적어도 되게" 해주는 단짝이에요.

설정(Settings) — 처음 한 번은 꼭

왼쪽 아래 내 이름/프로필 → 설정(Settings)으로 들어가요. 딱 네 가지만 손보면 끝.

① 프로필 / 이름

Claude가 나를 어떻게 부를지, 내 직업이 뭔지 적어두는 곳. "초등학교 교사"라고만 적어도 답이 맞춤형이 돼요.

② 화면(Appearance) — 다크모드 / 글자

눈이 편한 다크 모드로 바꾸거나 밝기를 고를 수 있어요. 언어가 영어로 보이면 한국어로 바꾸세요.

③ 데이터·개인정보 (가장 중요)

🔐 모델 학습에 내 대화 쓰기 — 끄는 걸 추천

설정의 개인정보(Privacy) 항목에 "내 대화를 모델 개선에 사용" 같은 스위치가 있어요. 교사 업무엔 민감 정보가 많으니 끄기(Off)를 권장해요. (계정·지역에 따라 항목 이름이 조금 다를 수 있어요.)

④ 기능 / 베타 (Feature Preview)

아티팩트·스킬·커넥터 같은 새 기능을 켜고 끄는 곳. 안 보이는 기능이 있으면 여기서 켜져 있는지 확인하세요.

✅ 처음 5분 체크리스트

① 언어 한국어 ② 직업 "교사" 입력 ③ 데이터 학습 끄기 ④ (눈 편하게) 다크모드 — 이 네 가지만 해두면 준비 끝!

다음: 지침 ↓
7

지침 — "나에 대해 알려주기"

한 번 적어두면 모든 대화에 자동 적용. 매번 같은 설명 반복 안 해도 돼요.

설정 안에 "개인 설정 / Claude에게 알려줄 내용(Personal preferences)" 같은 칸이 있어요. 여기에 나와 내 업무를 적어두면, Claude가 늘 그걸 알고 답해요. ChatGPT의 "Custom Instructions"와 같은 거예요.

교사용 지침 예시 — 그대로 복붙해도 OK

붙여넣기용 저는 대한민국 초등학교 교사예요. 학생·학부모 대상 글이 많습니다.
- 항상 존댓말과 정중한 톤을 써주세요.
- 가정통신문·안내문은 격식 있고 따뜻하게.
- 어려운 용어는 쉽게 풀어서 설명해주세요.
- 답이 길면 끝에 한 줄 요약을 붙여주세요.
- 학생 실명·개인정보가 보이면 가명 처리를 먼저 권해주세요.
💡 활용

학년이 바뀌거나 업무가 바뀌면 이 내용만 고치면 돼요. 특정 업무 전용 설정이 필요하면 → 9번 프로젝트에서 프로젝트별 지침을 따로 둘 수 있어요.

다음: 아티팩트 ↓
8

아티팩트(Artifacts) — 옆 화면의 결과물

긴 결과물을 채팅 속이 아니라 오른쪽 별도 창에 깔끔하게 만들어주는 기능.

문서, 표, 학습지, 안내문, 심지어 간단한 웹페이지나 미니 도구를 만들면 Claude가 오른쪽에 아티팩트 패널을 열어 보여줘요. 채팅으로 스크롤 안 해도 결과물만 따로 보고 고칠 수 있어요.

이렇게 만들어져요

  1. 그냥 부탁하면 자동 생성 "○○ 학습지를 만들어줘", "이걸 표로 정리해줘"라고 하면 오른쪽에 결과물 패널이 떠요.
  2. 바로 수정 요청 "글자 더 크게", "2번 문제 바꿔줘"처럼 말하면 그 패널이 즉시 업데이트돼요.
  3. 복사·다운로드 패널 위 버튼으로 복사하거나 내려받아 한글·워드에 붙여넣기.
  4. 공유(선택) 만든 결과물을 링크로 동료 교사에게 공유할 수도 있어요.

프로젝트(Projects) — 업무별 작업방

관련 대화 + 참고자료 + 전용 지침을 한 폴더(작업방)에 모아두는 기능. (주로 Pro 이상)

예를 들어 "3학년 2반 학급경영" 프로젝트를 만들어두면, 그 안의 모든 대화가 같은 자료와 같은 규칙을 공유해요. 매번 배경 설명을 안 해도 돼요.

만드는 법

  1. 왼쪽 사이드바 → 프로젝트 → ＋ 새 프로젝트
  2. 이름 짓기 예: "생기부 작성", "학부모 소통", "수업 자료 만들기"
  3. 지식(자료) 올리기 학급 규칙, 작년 양식, 학교 서식 등 PDF·문서를 넣어두면 Claude가 늘 참고해요.
  4. 프로젝트 지침 적기 "이 방에선 항상 ○○ 양식으로, 존댓말로" 같은 전용 규칙을 따로 지정.
  5. 그 안에서 대화 시작 이제 새 대화를 열면 자료·지침이 자동 적용돼요.
💡 지침(7번)과 차이

지침 = 모든 대화에 적용되는 "나"의 기본 정보. 프로젝트 = 특정 업무방에만 적용되는 자료+규칙. 업무가 여러 개면 프로젝트로 칸막이를 치세요.

🔐 자료 올릴 때

학생 명렬표·성적처럼 민감한 파일은 이름을 가리거나 번호로 바꾼 뒤 올리세요. 자세한 건 맨 아래 개인정보 주의에서.

스킬 & MCP(커넥터) — 한 단계 더

조금 고급이지만, 한 번 보면 "아 이런 게 있구나" 싶을 거예요. 안 써도 기본 기능엔 문제없어요.

스킬(Skills)이란?

특정 작업을 잘하도록 미리 훈련된 묶음 기능이에요. 예: "워드 문서 만들기", "엑셀 정리", "PPT 만들기" 스킬을 켜두면 그런 요청을 훨씬 깔끔하게 해줘요. 사람으로 치면 '전문 자격증'을 추가로 달아주는 것과 비슷해요.

  1. 설정 → 기능/스킬(Capabilities·Skills) 메뉴로 이동
  2. 원하는 스킬 켜기(On) 예: 문서(docx)·스프레드시트(xlsx)·발표자료(pptx)
  3. 평소처럼 부탁 "이 표를 엑셀 파일로 만들어줘"라고 하면 스킬이 알아서 작동해 파일을 만들어줘요

MCP(커넥터)란?

Claude를 내 다른 앱과 연결해주는 기능이에요. 연결해두면 "내 구글 드라이브에서 그 문서 찾아줘", "다음 주 시간표 캘린더에 넣어줘"처럼 시킬 수 있어요. 한마디로 Claude에게 손발을 달아주는 것.

연결 예시이렇게 시킬 수 있어요
구글 드라이브"드라이브에서 지난 회의록 찾아 요약해줘"
구글 캘린더"이 행사 일정들 캘린더에 등록해줘"
지메일"이 학부모 메일에 정중한 답장 초안 써줘"

커넥터 연결하기 (따라하기)

  1. 설정 → 커넥터(Connectors) 메뉴 열기
  2. 연결할 앱 선택 → "연결(Connect)" 예: Google Drive
  3. 구글 로그인 & 권한 허용 팝업에서 어떤 권한을 주는지 읽고 허용
  4. 대화에서 사용 이제 "내 드라이브에서 ○○ 찾아줘"라고 하면 Claude가 실제로 찾아봐요
⚠️ 권한은 신중하게

커넥터는 내 계정 자료에 접근해요. 꼭 필요한 것만 연결하고, 학교 공용·민감 계정은 피하세요. 안 쓰면 같은 메뉴에서 언제든 연결 해제할 수 있어요.

💡 초보자에게

스킬·MCP는 "있다는 것만" 알아두고, 1~9번에 익숙해진 다음 천천히 도전해도 충분해요. 급할 거 없어요!

바로 써먹는 교사 활용 예시

복사해서 붙여넣고, 괄호 부분만 바꿔보세요.

가정통신문 (현장체험학습) 안내 가정통신문을 정중한 톤으로 써줘. 날짜 (5/20), 장소 (○○), 준비물 (○○), 회비 (○○원). A4 반 장 분량.
생기부 문구 다음 학생 관찰 메모를 생활기록부 '행동특성 및 종합의견' 문장으로 다듬어줘. 과장 없이 사실 기반, 긍정적 표현으로. 메모: (붙여넣기)
학부모 답장 학부모님께 받은 다음 메시지에 정중하고 공감 어린 답장 초안을 써줘. 너무 길지 않게. 메시지: (붙여넣기)
시험·학습지 초등 (5)학년 (수학 분수) 단원 형성평가 문제 (10)개 만들어줘. 난이도 섞고, 정답과 간단 해설도 따로.
수업 아이디어 (4)학년 (환경) 주제로 40분 수업 활동 3가지 제안해줘. 준비물 적고, 모둠활동 포함.
행정·요약 첨부한 공문(PDF)에서 우리 반이 해야 할 일과 마감일만 표로 정리해줘.

꼭 지켜요 — 학생 개인정보 주의

AI는 편리하지만, 교사는 민감정보를 다루는 직업이에요. 이 한 섹션은 꼭 읽어주세요.

🔐 넣지 말아야 할 것

학생·학부모 실명, 주민번호·생년월일, 연락처·주소, 성적 원본, 건강·가정사 등 민감정보를 그대로 입력하지 마세요.

  • 가명·번호로 바꾸기 "김민준" → "학생A", 반 전체는 "1번, 2번…"으로.
  • 파일도 마찬가지 명렬표·성적표를 올릴 땐 이름 칸을 가리거나 번호로 치환 후 올리기.
  • 데이터 학습 끄기 6번 설정에서 "모델 개선에 대화 사용"을 꺼두세요.
  • 결과물 재확인 AI가 만든 생기부·공문은 반드시 사람이 검토 후 사용.
  • 학교·교육청 지침 우선 소속 기관의 생성형 AI 사용 지침이 있다면 그게 우선이에요.
